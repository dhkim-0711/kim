# 산업진흥법 한눈에 보기 (정적 페이지)

`index.html`을 더블클릭해서 브라우저로 열면 됩니다(서버 불필요).

## 기능

- 검색: 법명/부처/산업/키워드(지원수단 포함) 검색
- 필터: **산업**, **지원수단**(태그) 필터
- 보기: **카드** / **표** 전환
- 즐겨찾기: 항목 우측의 `☆/★` (브라우저 로컬 저장)
- 내보내기: 현재 필터 결과를 CSV로 저장

## 데이터 수정

`laws.data.js`의 `LAWS` 배열을 편집하세요.

각 항목 필드:

- `id` (필수): 고유 문자열 (즐겨찾기 저장 키)
- `title` (필수): 법/정책명
- `industry`: 산업 분류
- `ministry`: 소관부처/기관
- `instruments`: 지원수단 태그 배열 (예: `["R&D","인력","세제"]`)
- `support`: 핵심지원 한 줄 요약
- `scope`: 적용/대상
- `note`: 메모(선택)
- `links`: `{ title, url }` 배열 (선택)

## 권장 확장

- 법령/의안/보도자료의 **정확한 공식 링크**를 `links`에 채우기
- “유사도” 기준(예: 지원수단 교집합)으로 자동 정렬
- 항목을 클릭하면 상세 모달(조문/사업/예산/근거조항) 표시
